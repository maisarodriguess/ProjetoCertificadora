Planejamento do Projeto REA

Projeto Certificadora

Introdução ao Front-End: HTML, CSS e JavaScript na prática

Proposta

Este projeto tem como objetivo desenvolver um Recurso Educacional Aberto (REA) que ensina, de forma simples e prática, os fundamentos do desenvolvimento Front-End, usando as linguagens HTML, CSS e JavaScript. O conteúdo será voltado para iniciantes na área de programação e será apresentado em formato de vídeo com demonstrações práticas.

Formato do Material

  • Vídeo explicativo (com narração e demonstração de código)
  • Repositório com os arquivos do projeto (HTML, CSS, JS)
  • Página online publicada com GitHub Pages

Tema / Disciplina Envolvida

Programação Web Front-End / Fundamentos de Programação / Lógica de Programação

Tecnologias Utilizadas

  • HTML5
  • CSS3
  • JavaScript (básico)
  • GitHub e GitHub Pages

Estrutura do Website

O site será estruturado em uma única página, dividida em três seções principais: Apresentação do curso, Conteúdo e Contatos. Os usuários poderão navegar entre essas seções tanto rolando a página quanto utilizando uma barra de navegação (navbar) no topo da página.

Apresentação

Esta seção dará uma breve apresentação do curso.

Conteúdo

Aqui, os usuários poderão acessar os vídeos do curso. Esta seção será organizada em módulos de forma a facilitar a navegação e o acesso ao conteúdo.

Contatos

Esta seção fornecerá os meios de contato dos autores do projeto, como os links para seus perfis no LinkedIn e GitHub.
